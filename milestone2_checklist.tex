\section{Project Overview}
This milestone focuses on completing a reliable end-to-end study loop: deterministic trial generation, accurate gaze-aware vs. gaze-unaware assistance, and complete post-trial data collection/visualization for analysis readiness.

\section{Milestone 2 Deliverables}

\begin{enumerate}[leftmargin=0.8cm, itemsep=0.45cm, label=\textbf{\arabic*.}]
    \item \taskbox\ Build a virtual shelf anchored above the real table with stable object placement and readable goal UI surfaces \hfill {\color{successgreen}\bfseries [12 points]}

    \item \taskbox\ Create a deterministic challenge generation pipeline (fixed seed, reproducible rounds, and reproducible object placements) \hfill {\color{successgreen}\bfseries [12 points]}

    \item \taskbox\ Implement round-state timing flow: fixation cross, transition display, randomized blank pause, then spawn/search \hfill {\color{successgreen}\bfseries [10 points]}

    \item \taskbox\ Alternate condition assignment by round (gaze-aware vs. gaze-unaware) with explicit runtime state tracking/debug display \hfill {\color{successgreen}\bfseries [10 points]}

    \item \taskbox\ Improve gaze-contingent hint reliability so spatial guidance maps correctly to user gaze and target location \hfill {\color{successgreen}\bfseries [12 points]}

    \item \taskbox\ Refine voice agent behavior: interrupt speech at round end, announce next-round goal during transition, and remove repeated phrasing \hfill {\color{successgreen}\bfseries [10 points]}

    \item \taskbox\ Integrate NASA-TLX submission at session end and persist responses with participant/session identifiers \hfill {\color{successgreen}\bfseries [10 points]}

    \item \taskbox\ Logging of data for analysis \hfill {\color{successgreen}\bfseries [12 points]}

    \item \taskbox\ Present an end-of-session participant summary (e.g., completion time, per-round performance, hint usage trends) \hfill {\color{successgreen}\bfseries [6 points]}

    \item \taskbox\ Basic charting of data for presentation and sanity checks (e.g., bar charts for completion time/error by condition) \hfill {\color{successgreen}\bfseries [6 points]}
\end{enumerate}

\vspace{0.4cm}
\textbf{Total: 100 points}
